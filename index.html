PROJECTS

Amplifier Design Project

The project was created in MultiSim. It involved designing and simulating a single-supply multistage, inverting, and transistor amplifier with various specifications. Results were simulated through waveforms and those waveforms were further analyzed to match theoretical values. Minor discrepancies were present however the construction of the Amplifier was successful.

General Purpose Processor

The project was created in Quartus. It consisted of designing a general purpose processor with three different Arithmetic and Logic Units(ALU). They were created using components including latches, 4-16 decoder, Moore/Finite State Machine(MSM/FSM). Results were displayed on seven-segment displayes (SSEG) and were plotted on waveforms.

BookStore Application

The project was coded in NetBeans 8.2 and had a responsive UI with the help of JavaFX. It incorporated different windows in which various operations were performed. These included login, purchasing, and admin options which allowed you to modify the bookstore.
